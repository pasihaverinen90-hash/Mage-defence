# Mage Arena

A browser-based idle mage arena game built with Phaser 3, TypeScript, and Vite.

## Local development

```bash
npm install
npm run dev       # dev server at http://localhost:5173
```

## Build

```bash
npm run build     # production build → dist/
npm run preview   # preview the dist/ build locally
```

## Deploy

The build output in `dist/` is self-contained and suitable for GitHub Pages (uses relative asset paths via `base: './'`).
